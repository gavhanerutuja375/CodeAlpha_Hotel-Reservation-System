# CodeAlpha_Hotel-Reservation-System
---
## Description
This project is a simple **Java-based Hotel Reservation System** developed as part of the CodeAlpha Java Programming Internship.  
The program allows users to search available rooms, book rooms, cancel reservations, and view booking details.

---
## Features
- Search available rooms
- Room categories: Standard, Deluxe, Suite
- Book hotel rooms
- Cancel reservations
- Payment simulation
- Store booking details using File Handling
---
## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- File Handling
